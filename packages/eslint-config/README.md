# @hex-labs/eslint-config

### Installation

```bash
npm install --save-dev @hex-labs/eslint-config eslint

yarn install --dev @hex-labs/eslint-config eslint
```

### Usage

Add this line to your `package.json`

```json
"eslintConfig": {
  "extends": "@hex-labs/eslint-config"
}
```
